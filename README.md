# VaR Model Comparison & Backtesting Dashboard

An interactive market risk dashboard that compares four Value at Risk (VaR) estimation methods across historical stress periods, built with Python and Streamlit.

🔗 **[Live Demo](https://your-app-name.streamlit.app)** <!-- Replace with your Streamlit Cloud URL -->

---

## Overview

Value at Risk (VaR) is one of the most widely used measures in financial risk management — yet standard models are known to systematically underestimate tail risk during market stress. This project empirically demonstrates that failure and benchmarks four models of increasing sophistication against real crisis data.
